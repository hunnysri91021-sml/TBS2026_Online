แจ้งเตือนทุกครั้งที่มีการเปลี่ยนแปลงสถานะ
Apps Script จัดการส่ง LINE OA + Email แทน browser (เพราะ CORS) — ดูคู่มือการตั้งค่า →
💬
LINE OA (Messaging API)
⚠ LINE Notify ปิดให้บริการแล้ว (มี.ค. 2568) — ระบบนี้ใช้ LINE OA Messaging API แทน ต้องมี LINE Official Account
✅ Token ถูกเก็บใน Apps Script Properties เท่านั้น และไม่แสดงบนหน้าเว็บเพื่อความปลอดภัย
ตั้งค่าได้ที่ Apps Script → Project Settings → Script Properties → LINE_CHANNEL_ACCESS_TOKEN
User ID: เริ่มด้วย U | Group ID: เริ่มด้วย C หรือ G — ดูได้ใน LINE OA Manager ↗
✉
Email แจ้งเตือน
📋 คู่มือตั้งค่า LINE Notify + Email
⚠ LINE Notify ปิดแล้วตั้งแต่ 31 มี.ค. 2568 — ต้องใช้ LINE OA Messaging API แทน
ขั้นตอน 1 — สร้าง LINE Official Account
  1. ไปที่ manager.line.biz ↗ → สร้าง LINE OA (ฟรี)
  2. ไปที่ developers.line.biz ↗ → สร้าง Provider → สร้าง Channel ประเภท Messaging API
  3. แท็บ Basic settings → Channel access token → Issue → Copy
  4. วาง token ในช่อง Channel Access Token
  5. หา User ID / Group ID ที่ต้องการส่ง → วางในช่อง To
ขั้นตอน 2 — สร้าง Google Apps Script สำหรับส่ง
  1. ไปที่ script.google.com → New Project
  2. วางโค้ด Apps Script จากไฟล์ TBS2026_Notify.gs
  3. Deploy → Web App → Execute as: Me → Anyone → Copy URL
  4. วาง URL ในช่อง Google Apps Script URL ด้านบน
⚠ LINE Notify ต้องส่งผ่าน Apps Script เพราะ browser block Cross-Origin request ไปหา LINE โดยตรง
👤 จัดการพนักงาน (System Admin)
เพิ่ม / ลบพนักงานได้เฉพาะ System Admin เท่านั้น
ⓘ กรอกข้อมูลพนักงานแล้วกด เพิ่ม/อัปเดต หากใช้รหัสเดิม ระบบจะอัปเดตข้อมูลเดิม และสร้างสถานะเริ่มต้นเป็น On ในช่วงเวลาปัจจุบัน
🏢 พนักงานในระบบ
📝 สรุปรายการที่แก้ไข
สรุปสถานะการเข้างานที่ถูกแก้ไข เลือกช่วงเวลาได้ แยกตามฝ่าย และส่งอีเมลได้
ⓘ ข้อมูลดึงจาก Google Sheet โดยตรง — รายการที่แสดงคือสถานะที่ถูกบันทึก/แก้ไขในช่วงเวลาที่เลือก จัดกลุ่มแยกตามฝ่าย พร้อมส่งสรุปทางอีเมลให้ผู้บริหารหรือหัวหน้าฝ่ายได้
✉ ส่งสรุปทางอีเมล
ส่งผ่าน Google Apps Script เดียวกับระบบแจ้งเตือน — ตั้งค่า URL ได้ที่แท็บ System Admin · ไฟล์ Excel สร้างจากเบราว์เซอร์แล้วแนบไปกับเมล
